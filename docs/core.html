01

Ինչպե՞ս է այն աշխատում

Ժամանակակից JS-ը օգտագործում է JIT (Just-In-Time) կոմպիլյացիա։ Սա նշանակում է, որ կոդը թարգմանվում է մեքենայական լեզվի հենց կատարման պախին՝ ապահովելով բարձր արագություն։

V8 Engine (Chrome/Node)

Ամենաարագ շարժիչներից մեկը, որը փոխակերպում է JS կոդը օպտիմիզացված մեքենայական կոդի։

Single Threaded

JS-ը կատարում է մեկ գործողություն մեկ պահին, բայց օգտագործում է Event Loop՝ ասինխրոնության համար։
