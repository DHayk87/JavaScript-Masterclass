Promises (Խոստումներ)

"Ասինխրոն գործողությունների կառավարման ժամանակակից գործիքը։"

01

Promise-ի 3 վիճակները

Promise-ը միշտ գտնվում է այս 3 վիճակներից մեկում.

  • Pending (Սպասում): Սկզբնական վիճակը, երբ գործողությունը դեռ չի ավարտվել։
  • Fulfilled / Resolved (Կատարված): Գործողությունը հաջողությամբ ավարտվել է։
  • Rejected (Մերժված): Տեղի է ունեցել սխալ։

02

Ստեղծում և Օգտագործում

Promise ստեղծելու համար օգտագործում ենք new Promise կոնստրուկտորը, որը ստանում է երկու ֆունկցիա՝ resolve և reject:

const myPromise = new Promise((resolve, reject) => {
  const success = true;

  setTimeout(() => {
    if (success) {
      resolve("Տվյալները հաջողությամբ բեռնվեցին:");
    } else {
      reject("Սխալ տեղի ունեցավ:");
    }
  }, 2000);
});

// Օգտագործումը
myPromise
  .then(data => console.log(data))   // Եթե resolve եղավ
  .catch(err => console.error(err)) // Եթե reject եղավ
  .finally(() => console.log("Գործողությունն ավարտվեց:"));
03

Promise Chaining (Շղթայական կանչեր)

Սա Promise-ի ամենահզոր կողմն է։ Մենք կարող ենք կապել մի քանի .then() իրար, և ամեն հաջորդը կստանա նախորդի վերադարձրած արժեքը։

fetch('https://api.example.com/user/1')
  .then(response => response.json()) // Վերադարձնում է նոր Promise
  .then(user => user.name)           // Վերցնում ենք անունը
  .then(name => console.log(name))   // Տպում ենք
  .catch(err => console.log(err));   // Մեկ catch-ը բավարար է ամբողջ շղթայի սխալները բռնելու համար
