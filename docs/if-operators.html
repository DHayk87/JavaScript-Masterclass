Պայմաններ և Լոգիկա

"Ծրագրի ընթացքը կախված է ճիշտ ընտրությունից և լոգիկական կապերից:"

01

Համեմատության Օպերատորներ

Համեմատության օպերատորները վերադարձնում են բուլյան արժեք՝ true կամ false:

Հիմնական Օպերատորներ

  • a > b, a < b (Մեծ, փոքր)
  • a >= b, a <= b (Մեծ կամ հավասար)
  • a == b, a != b (Հավասարություն)

Խիստ Հավասարություն

=== օպերատորը ստուգում է թե՛ արժեքը, թե՛ տիպը: Խորհուրդ է տրվում միշտ օգտագործել այն:

alert( 2 > 1 ); // true
alert( '2' == 2 ); // true (ոչ խիստ)
alert( '2' === 2 ); // false (խիստ)
02

Պայմանական Ճյուղավորում (if, else if, else)

if հրահանգը ստուգում է պայմանը և կատարում կոդը, եթե արդյունքը true է: Բարդ դեպքերում օգտագործվում են else if և else:

if-else if-else Կառուցվածք

if (time < 12) {
  alert('Բարի լույս');
} else if (time < 18) {
  alert('Բարի օր');
} else {
  alert('Բարի երեկո');
}

Տերնար օպերատոր (Ternary Operator)

Կարճ տարբերակ պայմանի համար (Conditional Operator):

let result = (age >= 18) ? "Թույլատրված" : "Մերժված";

Կառուցվածք՝   պայման ? արժեք_եթե_true : արժեք_եթե_false

03

Լոգիկական Օպերատորներ (OR, AND, NOT)

|| (OR / ԿԱՄ)

Ճիշտ է, եթե գոնե մեկ պայմանը ճիշտ է:

true || false; // true

&& (AND / ԵՎ)

Ճիշտ է միայն այն դեպքում, եթե բոլորը ճիշտ են:

true && false; // false

! (NOT / ՈՉ)

Շրջում է արժեքը:

!true; // false

?? (Nullish Coalescing)

Վերադարձնում է առաջին «գոյություն ունեցող» արժեքը:

result = a ?? b;
