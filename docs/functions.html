Ֆունկցիաներ (Functions)

"Ֆունկցիաները ծրագրի հիմնական կառուցողական բլոկներն են:"

01

Function Declaration (Ֆունկցիայի հայտարարում)

Սա հանդիսանում է HoistableDeclaration: JS շարժիչը Environment Record-ը ստեղծելիս այս ֆունկցիաները միանգամից սկզբնավորում է (initialize):

Սինտաքս

function sayHi() {
  console.log("Ողջույն");
}

Առավելությունը

Կարելի է կանչել կոդի ցանկացած հատվածից (նույնիսկ հայտարարումից առաջ), ինչը թույլ է տալիս կառուցել կոդը «կարևորից դեպի մանրուքներ» սկզբունքով:

02

Function Expression (Ֆունկցիոնալ արտահայտություն)

Այստեղ ֆունկցիան հանդես է գալիս որպես արժեք: Այն գրանցվում է որպես լեքսիկական փոփոխական և ենթարկվում է let/const կանոններին:

Սինտաքս

let sayHi = function() {
  console.log("Ողջույն");
};

Տարբերությունը

Քանի դեռ ծրագրի կատարումը չի հասել այս տողին, ֆունկցիան գտնվում է Temporal Dead Zone-ում և հասանելի չէ:

03

Arrow Functions (Սլաքային ֆունկցիաներ)

ES6-ի ամենակարևոր լրացումներից մեկը: Սրանք պարզապես «կարճ գրելաձև» չեն, այլ ունեն էական ֆունկցիոնալ տարբերություններ:

Սինտաքս

const sum = (a, b) => a + b;

Ինչու՞ են հատուկ

  • Lexical this: Չունեն սեփական this: Վերցնում են այն ստեղծման միջավայրից:
  • Constructor: Չեն կարող օգտագործվել new օպերատորի հետ:
  • Arguments: Չունեն սեփական arguments օբյեկտը:
04

IIFE (Անմիջապես կանչվող ֆունկցիաներ)

Չնայած ժամանակակից JS-ում ունենք Block Scope, IIFE-ն դեռևս օգտագործվում է որոշակի դեպքերում:

Սինտաքս

(function() {
  let privateVar = "Գաղտնի տվյալ";
})();
