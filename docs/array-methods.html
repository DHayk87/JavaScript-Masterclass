Method Description Return Value Mutates?
push() Ավելացնում է տարր վերջից Նոր length-ը Yes
pop() Հեռացնում է վերջին տարրը Հեռացված տարրը Yes
shift() Հեռացնում է առաջին տարրը Հեռացված տարրը Yes
unshift() Ավելացնում է տարր սկզբից Նոր length-ը Yes
slice() Կտրում է զանգվածի մի մասը Նոր պատճենված զանգվածը No
splice() Ավելացնում/հեռացնում է տարրեր ցանկացած դիրքից Հեռացված տարրերի զանգվածը Yes
