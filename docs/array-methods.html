Արտադրողականության աղյուսակ

Method Time Complexity (Big O) Mutates? ↕
push() O(1) (Amortized constant time) Yes
pop() O(1) (Constant time) Yes
shift() O(n) (Must re-index all remaining elements) Yes
unshift() O(n) (Must re-index all elements) Yes
slice() O(k) (Copies `k` extracted elements) No
splice() O(n) (Must shift trailing elements) Yes
sort() O(n log n) (Standard sorting complexity) Yes
indexOf() / includes() O(n) (Linear search) No
find() / some() / every() O(n) (Linear scan with early exit) No
map() / filter() / forEach() O(n) (Must visit every element) No
reduce() / reduceRight() O(n) (Linear accumulation) No
flat() / flatMap() O(n + k) (Linear plus depth overhead) No
toReversed() / toSorted() / toSpliced() O(n) / O(n log n) (Immutable versions) No
copyWithin() / fill() O(n) (Linear modification) Yes
