## 📁 Նախագծի Կառուցվածքը (Architecture)

```text
JS-Docs/
├── docs/                 # Հիմնական փաստաթղթավորում (HTML)
├── public/               # Հանրային ռեսուրսներ
│   ├── images/           # Նկարներ և պատկերներ
│   └── styles/           # Global CSS (style.css)
├── templates/            # HTML Կաղապարներ
├── guides/               # Ուղեցույցներ և Style Guide
└── node_modules/         # Արտաքին կախվածություններ (Mermaid.js)
```
